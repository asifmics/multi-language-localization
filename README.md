
## Project setup
```
composer install
php artisan key:generate

```
#### Change env file

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=root
```


#### Run this command
```
php artisan optimize:clear
php artisan migrate:fresh --seed

npm install
npm run dev

```

#### Default User

```
test@example.com
password
```

#### Laravel localization use for run this command
```
php artisan lang:publish
```
